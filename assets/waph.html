Repository organization:

  • Labs: Hands-on exercises in lectures
  • Lab 1: Foundations of the Web
    The lab is divided into two sections: web application programming and web and HTTP Protocol. We examined the HTTP protocol on the Web and HTTP protocol using Wireshark. The http request and http response are the two components of the synchronous http protocol. The browser contacts the server with an HTTP request and then waits for a response.
  • Lab 2: Front-end Web Development
    We explored real-world exercises including HTML, JavaScript, Ajax, CSS, jQuery, and Web API connection in Lab 2. The main goal was to apply theoretical knowledge from lectures to practical work in order to reinforce it. The lab's objectives were to improve our knowledge of web development technologies and expand on how the many components of creating dynamic web apps work together. The lab will provide you with hands-on experience in basic HTML structure creation, JavaScript implementation, Ajax integration for asynchronous communication, CSS styling of web pages, jQuery simplification of scripting, and integration of external Web APIs for dynamic content fetching and display. The lab also focused on using Markdown for efficient reporting and documentation procedures.

  • Hackations
  • Hackathon 1: Cross-site Scripting Attacks and Defenses
    The "Cross-site Scripting Attacks and Defenses" hackathon provides a deep dive into the complex realm of web application security, with a specific emphasis on addressing the widespread danger posed by cross-site scripting (XSS) attacks. XSS vulnerabilities pose a substantial threat to web applications by enabling nefarious individuals to inject and activate scripts within legitimate web pages, posing risks to user data, session cookies, and the overall integrity of systems. This event is designed to furnish participants with a thorough grasp of XSS vulnerabilities, including how they are exploited and the implementation of robust defense strategies to mitigate these risks effectively.
